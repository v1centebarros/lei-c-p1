#Exemplo 4 - Definir no Labirinto um spot com uma altura ou o raio igual a zero.

FICHEIRO:

    LABIRINTO (NAME = "teste" WIDTH = 17 HEIGHT = 14) { 
        BEACON = (3,1) HEIGHT = 3;
        SPOT = (26,7) HEIGHT = 0 RADIUS = 1;
    };

TERMINAL:
    [Line 3] NameError: A Height do Spot tem ser superior a zero.
FICHEIRO:

    LABIRINTO (NAME = "teste" WIDTH = 17 HEIGHT = 14) { 
        BEACON = (3,1) HEIGHT = 3;
        SPOT = (26,7) HEIGHT = 2 RADIUS = 0;
    };

TERMINAL:
    [Line 3] NameError: O Radius do Spot tem ser superior a zero.
